Test NamePnC with all valid certificates
Test IdentifierITP-03-14-001
Test TypeBasic Certificate Validity Tests
Test CategorySingle PKI Tests
Test PurposeTo ensure that the EV and EVSE can establish a secure communication session using valid PKI certificates following the ISO 15118 standard.
Observed MetricsSession Initialization, Messages, EV UI response, EVSE UI response, HLC Message Sequence
Intended MRECs/Errors--
Other Possible MRECs--
Test ConditionsAuthentication Type/sPnC
HLC Protocol/sISO 15118-2, ISO 15118-20
Involved System/sEV, EVSE, PKI Provider
Certificate/s (PKI)Valid Certificate/sV2G Root CA Cert, EV Provisioning Cert, SECC Leaf Certificate, Contract Cert, (Optionally: EV SubCA1 Cert, EV SubCA2 Cert, CSO SubCA1 Cert, CSO SubCA2 Cert, ProvServ Cert)
Invalid Certificate/sNone
Steps1. Ensure 'Valid Certificate/s" are installed on the appropriate entity within the charging environment.
2. Ensure all certificates form a valid chain to a common trust anchor.
3. Set EVSE HLC highest priority to 'HLC Protocol'.
4. Set EVSE authentication option to 'Authentication Type'.
5. Plug-in vehicle.
6. Observe the session initialization into power transfer.
7. Terminate charge session 30-60s into power transfer.
8. Unplug EV.
Pass Criteria (Check box if met)
• EVSE validates Leaf certificates successfully, accepts valid EV certificates, and the charging session starts.
• EV validates and accepts EVSE certificates, and the charging session starts.
