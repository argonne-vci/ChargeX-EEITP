Test NameStart charge with different EV State of Charge [Multiple enumerations]
Test IdentifierITP-01-04-007
Test TypePower Transfer (PT) Tests
Test CategoryBaseline Functionality & Safety Tests
Test PurposeTo ensure charge session operates with EV at different starting SoC. Also to test that charge session does not occur when start SoC = target end SoC.
Observed MetricsBattery SoC [%], Voltage [V(DC)], Current [A(DC)], Battery Temp [°C], Cable Temp [°C], Connector temp [°C]
Intended MRECs/Errors1. -- 2. -- 3. --
Other Possible MRECs1. -- 2. -- 3. --
Test ConditionsEV Initial SoC1. <10%
2. >90%
3. 100%
EV SoC Charge Limit100%
HLC Protocol/sDIN SPEC 70121, ISO 15118-2, ISO 15118-20, J3400-1
Involved System/sEV, EVSE
Steps1. Set EVSE authentication option to Any.
2. Charge or discharge EV to 'EV State of Charge'.
3. Plug-in EV, authorize charge, observe power transfer.
4. If applicable, terminate the charge session 30-60 seconds into the power transfer.
5. Unplug EV.
Pass Criteria (Check box if met)
• EV successfully runs charge session at all starting EV SoCs, except when start SoC = target end SoC.
