Test NameManually request negative current to determine if allowed from EV
Test IdentifierITP-01-04-013
Test TypePower Transfer (PT) Tests
Test CategoryBaseline Functionality & Safety Tests
Test PurposeTo ensure negative current request is not accepted.
Observed MetricsBattery SoC [%], Voltage [V(DC)], Current [A(DC)], EVSE Available Current [A(DC)], EV Current Req [A(DC)]
Intended MRECs/Errors --
Other Possible MRECsNegative current request
Test ConditionsEV Initial SoC<75%
EV SoC Charge Limit100%
HLC Protocol/sDIN SPEC 70121, ISO 15118-2, ISO 15118-20, J3400-1
Involved System/sEV, EVSE
Steps1. Set EVSE authentication option to Any.
2. Plug-in EV, authorize charge, observe power transfer.
3. Ensure EVSE available current exceeds or meets maximum rating of EV.
4. Manually request -10% EV rated max current.
5. If session does not terminate, manually terminate charge session 1-2 minutes into power transfer.
6. Unplug EV.
