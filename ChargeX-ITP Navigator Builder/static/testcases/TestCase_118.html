Test NamePnC with adapter
Test IdentifierITP-02-17-003
Test TypeAdapter Tests
Test CategoryAdvanced Charging Tests
Test Purpose - To test the interoperability of adapters - To ensure PnC authentication method functions using adapters
Observed MetricsSession initialization stages, EV UI response, EVSE UI response
Intended MRECs/Errors--
Other Possible MRECsInvalidSequence
Test ConditionsAuthentication Type/sPnC
Plug or Authenticate firstPlug
HLC Protocol/sDIN 70121, J1772, ISO 15118-2, ISO 15118-20
Involved System/sEV, EVSE, Adapters
Steps1. Set EVSE HLC highest priority to 'HLC Protocol'.
2. Set EVSE authentication option to 'Authentication Type'.
3. Plug adapter into vehicle inlet first.
4. Within 30seconds, Plug-in connector to adapter.
5. Observe the session initialization into power transfer.
6. After 30 seconds, terminate the charge session.
7. Unplug EV
Pass Criteria (Check box if met)
• EV and EVSE allow for adapter to begin a PnC session.
• eSDP recognizes adapter is being used.
