Test NameSudden plug disconnection and re-connection during Session Initialization (SI)
Test IdentifierITP-01-03-008
Test TypeSession Initialization (SI) Tests [from Authentication to Power transfer]
Test CategoryBaseline Functionality & Safety Tests
Test PurposeTo assess the impact of abrupt plug disconnection and reconnection on the Session Initialization (SI) process.
Observed MetricsError messages
Intended MRECs/Errors"ConnetorLockFailure", "BrokenLatch", "PilotFailure"
Other Possible MRECs--
Test ConditionsEV Initial SoC<75%
EV SoC Charge Limit100%
HLC Protocol/sDIN SPEC 70121, ISO 15118-2, ISO 15118-20
Involved System/sEV, EVSE
Steps1. Set EVSE HLC highest priority to Any 'HLC Protocol'.
2. Set EVSE authentication option to Any 'Authentication Type'.
3. Plug in the EV and authorize a charge.
4. Immediately prior to session initialization and plug latching, promptly disconnect and reinsert the plug.
5. Observe the EVSE UI.
6. If charging initiates, stop it after 10 sec.
