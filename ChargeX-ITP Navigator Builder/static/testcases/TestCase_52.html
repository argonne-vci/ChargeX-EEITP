Test NameStart charge with both ports requesting nameplate rated Max available power per port
Test IdentifierITP-02-05-004
Test TypeSimultaneous Single Dispenser Charging (Multiple EVs required)
Test CategoryAdvanced Charging Tests
Test PurposeTo ensure EVSE can produce the nameplate rated power when two EVs are requesting the max capable power rated per port.
Observed MetricsVoltage [V(DC)], EVSE UI Behavior, Power split between ports
Intended MRECs/Errors--
Other Possible MRECs--
Test ConditionsAuthentication Type/sFreeVendMode
Plug or Authenticate firstPlug
EV Initial SoC<10%
EV SoC Charge Limit100%
Temperature"Nominal" (23°C)
HLC Protocol/sDIN SPEC 70121, ISO 15118-2, ISO 15118-20
Involved System/sEV, EVSE
Steps1. Set EVSE authentication option to "Authentication Type".
2. Plug-in EV1 and EV2 simultaneously into port1 and Port2 of EVSE.
3. Observe the EVSE UI and perform necessary actions for authorization of the individual port.
4. Manually request 100% port-rated max current for each port from the EVs.
5. Observe the EVSE UI and power transfer.
6. Manually terminate the charge session 2-3 minutes into power transfer.
7. Unplug both EVs from their respective EVSE ports.
Pass Criteria (Check box if met)
• EVSE simultaneously delivers the max rated power per port.
